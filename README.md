# Home Office Ergonomics Self-Assessment Toolkit

This resource kit provides a practical self-assessment checklist and additional reading materials to help employees set up an ergonomic home office. It is based on established ergonomic principles from authoritative sources such as NIOSH, government health departments, and academic ergonomics guidelines.

## How to Use This Kit

1. **Self-Assessment Checklist**: Use the `Checklist.md` file to evaluate your current home office setup. Go through each section and check off items that you have correctly configured.
2. **Further Reading**: Refer to the `Further_Reading.md` file for links to detailed ergonomic guidelines and research.
3. **Continuous Improvement**: Revisit the checklist periodically, especially after any changes to your workstation or if you experience discomfort.

## Files

- [Checklist.md](Checklist.md) – Self-assessment checklist covering chair, desk, monitor, environment, and work habits.
- [Further_Reading.md](Further_Reading.md) – Curated list of authoritative resources for deeper understanding.
